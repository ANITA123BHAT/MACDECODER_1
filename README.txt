# MAC DECODER FOR DOWNLINK AND UPLINK

To compile this project, you will need the following tools installed on your system:

`g++`: This is the C++ compiler. It is usually included with the GNU Compiler Collection (GCC).
'make`: This is a build automation tool that reads `Makefile`s to determine how to build the project.

#compiling the project
To compile the project, open a terminal and navigate to the project's root directory. Then, run the following command

-'make'
this will compile the project and produce and produce executable file.

# run the peoject
After compiling the project, you can run it by executing the `MAC_DECODER` file.
`./main'

# instructions to execute
- 'make'
- './main'
